# RenderDocs Roadmap

This document outlines the planned features and improvements for RenderDocs SDKs and developer tools.

## Current Status

### SDKs
- [x] Node.js SDK (v1.0.0)
- [x] Python SDK (v1.0.0)
- [x] Java SDK (v1.0.0)

### Integrations
- [x] Zapier Integration
- [x] Make (Integromat) Integration
- [x] n8n Integration
- [x] Google Sheets Add-on

## Planned Features

### Q1 2025

#### SDK Improvements
- [ ] Batch document generation with progress callbacks
- [ ] Retry logic with exponential backoff
- [ ] Request/response logging for debugging
- [ ] TypeScript strict mode improvements (Node.js)

#### New SDKs
- [ ] Go SDK
- [ ] Ruby SDK
- [ ] PHP SDK

### Q2 2025

#### Template Features
- [ ] Template versioning support in SDKs
- [ ] Template preview generation
- [ ] Template variable validation

#### Developer Experience
- [ ] SDK metrics and analytics
- [ ] OpenTelemetry integration
- [ ] Better error messages with suggested fixes

### Q3 2025

#### Advanced Features
- [ ] Streaming document generation
- [ ] Multi-document ZIP generation
- [ ] PDF/A compliance options
- [ ] Digital signature support
